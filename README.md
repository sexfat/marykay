<h1 align="center"> marykay </h1>

### 專案安裝
```bash
# 進入專案 - 港版
$ cd marykay-hk

# or 台版
$ cd marykay-hk

# 安裝
$ npm install
```

### nuxt 指令
```bash
# 執行開發環境
$ npm run dev

# 產生靜態頁面 路徑: ./dist/
$ npm run generate

```
